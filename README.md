# CS-enum-examples

These include examples on how to use enums in C#.

Things related to enums include:

- How to declare an enum
- Different ways to access enum values
- Comparing enums
- Iterating over enums (using getNames() and getValues())

## Prerequisites

- Make sure you have Visual Studio installed.

## To Open

- Navigate to the **enum-example** folder.
- Click on the **enum-example.sln** file to open the solution.

## To Run

- Click on the green play button at the top of the Visual Studio solution. This will open a console that prints the values shown in the examples.
